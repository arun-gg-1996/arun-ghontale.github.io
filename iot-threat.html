Multi-tiered threat detection pipeline using Graph Neural Networks, RAG interface, and Apache Spark for comprehensive security analysis.

The Challenge

IoT devices are increasingly targeted by sophisticated cyber attacks, with traditional signature-based detection systems failing to identify zero-day threats and novel attack patterns. Organizations need real-time, intelligent threat detection that can adapt to evolving attack vectors.

The Solution

Developed a scalable multi-tiered threat detection system combining traditional machine learning with advanced deep learning architectures.

Data Processing Pipeline

Apache Spark • IoT-23 Dataset

  • Architected distributed processing pipeline using Apache Spark for scalable data ingestion
  • Processed IoT-23 dataset with millions of network traffic records
  • Implemented feature extraction for time-series network behavior analysis

Real-Time Classification

LightGBM • Gradient Boosting

  • Deployed LightGBM classifier achieving 96.57% accuracy for real-time log classification
  • Optimized for low-latency prediction with sub-second response times
  • Handled imbalanced dataset with careful sampling techniques

Zero-Day Anomaly Detection

Denoising Autoencoder • Unsupervised Learning

  • Implemented Denoising Autoencoder for unsupervised anomaly detection
  • Detected novel attack patterns not seen during training
  • Reduced false positive rate through ensemble methods

Network Context Analysis

Graph Convolutional Network

  • Built GCN for node classification achieving 98% accuracy in identifying malicious devices
  • Leveraged network topology and device relationship patterns
  • Incorporated temporal evolution of network connections

Synthetic Data Generation

Generative Adversarial Network

  • Addressed data imbalance with GAN-generated synthetic samples
  • Improved model robustness across underrepresented attack types
  • Validated synthetic data quality through statistical analysis

Interactive RAG Interface

Model Context Protocol • RAG

  • Deployed models as tools via MCP server for comprehensive analysis
  • Built interactive RAG interface for security analysts
  • Enabled natural language queries about threat patterns and recommendations
